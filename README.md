# weather-news

openweatherAPI を用いて作成したお天気サイト。


##  機能
札幌、新潟、東京、大阪、福岡の 
5つの都市の6日分の天気を表示。.    
また、取得したそれらの気温や湿度などの天気の情報を.    
比較してランキング形式で表示させています。

## 使用技術

node.js: 16.    
Django: 3.2.12.   
vue: 2.6.14.   

使用した API
openweatherAPI
